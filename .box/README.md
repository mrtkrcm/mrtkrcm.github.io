Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

## Run Storybook

```bash
npm run storybook
# or
yarn storybook
```

## Build Static Storybook

```bash
npm run build-storybook
# or
yarn build-storybook
```

## Static

```bash
npm run build-storybook
# or
yarn build-storybook
# then
cd storybook-static
now
```
